# 星渊遗城 Demo

一款银河科幻风横版动作游戏 Demo，基于 Phaser 3 开发。

## 在线游玩

👉 **[点击开始游玩](https://你的用户名.github.io/你的仓库名/)**

## 操作说明

| 按键 | 动作 |
|------|------|
| ← → | 移动 |
| Z / ↑ / 空格 | 跳跃 |
| 空中再按跳跃 | 二段跳 |
| X | 近战三连击 |
| C | 远程攻击（消耗能量）|
| Shift | 冲刺 |
| 靠近墙 + 方向键 | 墙滑 / 墙跳 |

## 关卡

1. **Room 1** — 教学区，学习基础操作  
2. **Room 2** — 跳跃平台挑战  
3. **Room 3** — 战斗区，消灭所有敌人  
4. **Room 4** — 获取相位冲刺能力  
5. **Room 5** — Boss 战（相位监察者）

## 技术栈

- **引擎**: Phaser 3.60
- **语言**: JavaScript ES Modules
- **特效**: 程序化粒子 / Squash&Stretch 动画 / 银河科幻风 UI
